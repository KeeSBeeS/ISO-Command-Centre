# ISO Admin Command Framework v2.6.10

## Purpose
Restores the Customers area to the left navigation.

## Included
- Customers menu item in the left sidebar.
- Customers CRUD screen.
- `/customers` routes.
- `/clients` compatibility alias to the customers list.
- `customers` database table.
- Existing `clients.view` and `clients.manage` permissions renamed/described as customer permissions.
- System Administrator permission re-sync.
- Footer/platform version update to `2.6.10`.

## Install
Upload the package over the current installation and run:

```text
https://isoadmin.co.za/updates/v2-6-10
```

Then open:

```text
https://isoadmin.co.za/customers
```
