# ISO Admin Command Framework v2.7.1

## Customer site contacts and delete controls

This update builds on v2.7.0.

### Added

- Contact people under each customer site/location.
- Contact types: Accounts, Stores, Foreman, Engineer and Other.
- Captured contact details: name, position/title, phone, mobile/WhatsApp, email, status and notes.
- Delete button for contact people under a site/location.
- Delete button for client sites/locations from the customer profile and site profile.
- Delete button for customers from the customer list and customer profile.
- New permissions:
  - `site_contacts.view`
  - `site_contacts.manage`

### Install

Upload the package and run:

`/updates/v2-7-1`
