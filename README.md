# ISO Command Centre

Laravel-based ISO Admin / Command Centre platform for `isoadmin.co.za`.

This repository baseline was loaded from the v2.6.11 update package.

## Current baseline

- Version: v2.6.11
- Framework target: Laravel 11.x
- PHP target: PHP 8.2+
- Deployment style: shared hosting ZIP upload / manual file replacement

## Notes

- Do not commit `.env`, live credentials, API keys, uploaded files, cache files, or vendor dependencies.
- Use version tags for stable releases.
- Future updates should be committed here before ZIP deployment packages are produced.
