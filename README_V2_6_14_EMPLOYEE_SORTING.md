# ISO Admin Command Framework v2.6.14

## Employees page sorting

This update improves `/employees` / `/public/employees` by adding sortable table headings.

### Added

- Sort by employee name.
- Sort by department.
- Sort by role.
- Toggle ascending / descending by clicking the same heading again.
- Preserve sorting through search, status filters and pagination.

### Install

Upload the package and run:

`/updates/v2-6-14`
